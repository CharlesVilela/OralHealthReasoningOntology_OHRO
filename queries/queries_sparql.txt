# Patients and Demographic Data

These CQs focus on patient profiles, demographic attributes, and grouping based on basic characteristics.

---

## CQ01. What is the average age of patients by lesion category?

```sparql
PREFIX owl: <http://www.w3.org/2002/07/owl#> 
PREFIX oral: <http://example.org/oral-cancer-ontology#> 

SELECT ?lesionType (AVG(?age) AS ?avgAge) 
WHERE { 
    ?patient a oral:Patient ; 
             oral:age ?age ; 
             oral:hasMedicalImage ?img . 

    ?img oral:hasClinicalDiagnosis ?lesion . 
    ?lesion a ?lesionType . 

    FILTER(?lesionType != owl:NamedIndividual && ?lesionType != oral:Lesion) 
} 
GROUP BY ?lesionType 
ORDER BY DESC(?avgAge)
```

## CQ02. What is the clinical and habit profile of each patient (age, sex, smoking, alcohol, betel quid)?

```sparql
PREFIX oral: <http://example.org/oral-cancer-ontology#>

SELECT
    ?patient
    ?id
    ?age
    ?gender
    (GROUP_CONCAT(DISTINCT STRAFTER(STR(?riskFactor),"#"); separator=", ") AS ?riskFactors)
WHERE {

    ?patient a oral:Patient .

    OPTIONAL { ?patient oral:patientID ?id . }
    OPTIONAL { ?patient oral:age ?age . }
    OPTIONAL { ?patient oral:gender ?gender . }

    OPTIONAL {
        ?patient oral:hasRiskFactor ?riskFactor .
    }
}
GROUP BY ?patient ?id ?age ?gender
ORDER BY ?patient
```

## CQ03. Which patients present multiple risk factors?

```sparql
PREFIX oral: <http://example.org/oral-cancer-ontology#>
PREFIX rdf: <http://www.w3.org/1999/02/22-rdf-syntax-ns#>

SELECT
       ?patientID
       (COUNT(DISTINCT ?riskFactor) AS ?numRisks)
       (GROUP_CONCAT(
            DISTINCT STRAFTER(STR(?riskFactorType),"#");
            separator=", "
        ) AS ?riskFactors)
WHERE {

    ?patient oral:hasRiskFactor ?riskFactor ;
             oral:patientID ?patientID .

    ?riskFactor rdf:type ?riskFactorType .
}
GROUP BY ?patientID
HAVING (COUNT(DISTINCT ?riskFactor) > 1)
ORDER BY DESC(?numRisks)
```

# Oral Lesions

These CQs focus on lesion taxonomy, categorization, and distribution.

---

## CQ04. What are the main categories of oral lesions represented in the ontology?

```sparql
PREFIX ex: <http://example.org/oral-cancer-ontology#>
PREFIX rdfs: <http://www.w3.org/2000/01/rdf-schema#>

SELECT DISTINCT ?category ?label
WHERE {
    ?category rdfs:subClassOf ex:Lesion .

    OPTIONAL {
        ?category rdfs:label ?label .
    }
}
ORDER BY ?category
```

## CQ05. Which lesions are associated with more than one patient?

```sparql
PREFIX oral: <http://example.org/oral-cancer-ontology#>

SELECT
       ?category
       (COUNT(DISTINCT ?patient) AS ?numPatients)
WHERE {

    ?image oral:belongsToPatient ?patient ;
           oral:hasCategory ?category .
}
GROUP BY ?category
HAVING (COUNT(DISTINCT ?patient) > 1)
ORDER BY DESC(?numPatients)
```

# Medical Images and Annotations

These CQs focus on exams, image files, and lesion annotations.

---

## CQ06. Which image annotations (lesions) are associated with each exam?

```sparql
PREFIX oral: <http://example.org/oral-cancer-ontology#>
PREFIX rdfs: <http://www.w3.org/2000/01/rdf-schema#>

SELECT DISTINCT
       ?patient
       ?image
       ?annotation
       (COALESCE(?label,
                 STRAFTER(STR(?lesion),"#")) AS ?lesionName)
       ?x
       ?y
       ?width
       ?height
       ?area
WHERE {

    ?image oral:hasAnnotation ?annotation ;
           oral:belongsToPatient ?patient .

    OPTIONAL {
        ?annotation oral:indicatesLesion ?lesion .
        OPTIONAL { ?lesion rdfs:label ?label . }
    }

    OPTIONAL {
        ?annotation oral:hasBoundingBox ?bbox .

        OPTIONAL { ?bbox oral:bb_x ?x . }
        OPTIONAL { ?bbox oral:bb_y ?y . }
        OPTIONAL { ?bbox oral:bb_width ?width . }
        OPTIONAL { ?bbox oral:bb_height ?height . }
        OPTIONAL { ?bbox oral:area ?area . }
    }
}
ORDER BY ?patient ?image ?annotation
```

# Risk Factors

These CQs focus on habits, history, and causative factors.

---

## CQ08. Which risk factors appear in patients who have a specific lesion?

```sparql
PREFIX owl: <http://www.w3.org/2002/07/owl#>
PREFIX oral: <http://example.org/oral-cancer-ontology#>
PREFIX rdf:  <http://www.w3.org/1999/02/22-rdf-syntax-ns#>

SELECT DISTINCT
       ?lesionType
       ?riskFactorType
WHERE {

    ?image oral:hasClinicalDiagnosis ?lesion ;
           oral:belongsToPatient ?patient .

    ?patient oral:hasRiskFactor ?riskFactor .

    ?lesion rdf:type ?lesionType .
    ?riskFactor rdf:type ?riskFactorType .

    FILTER(?lesionType != owl:NamedIndividual)
    FILTER(?riskFactorType != owl:NamedIndividual)
}
ORDER BY ?lesionType ?riskFactorType
```

## CQ09. Which patients have risk factors but no recorded diagnosis?

```sparql
PREFIX oral: <http://example.org/oral-cancer-ontology#>

SELECT
       ?patient
       ?patientID
       (GROUP_CONCAT(DISTINCT STRAFTER(STR(?riskFactor),"#"); separator=", ") AS ?riskFactors)
WHERE {

    ?patient a oral:Patient ;
             oral:hasRiskFactor ?riskFactor .

    OPTIONAL {
        ?patient oral:patientID ?patientID .
    }

    FILTER NOT EXISTS {
        ?patient oral:hasMedicalImage ?image .
        ?image oral:hasClinicalDiagnosis ?diagnosis .
    }
}
GROUP BY ?patient ?patientID
ORDER BY ?patientID
```

## CQ12. Which risk factors are most frequently associated with malignant lesions?

```sparql
PREFIX oral: <http://example.org/oral-cancer-ontology#>
PREFIX rdf: <http://www.w3.org/1999/02/22-rdf-syntax-ns#>

SELECT
    ?lesionType
    ?riskFactorType
    (COUNT(DISTINCT ?patient) AS ?frequency)
WHERE {

    ?image oral:belongsToPatient ?patient ;
           oral:hasClinicalDiagnosis ?lesion .

    ?patient oral:hasRiskFactor ?riskFactor .

    ?lesion rdf:type ?lesionType .
    ?riskFactor rdf:type ?riskFactorType .

    FILTER(?lesionType != oral:Lesion)
}
GROUP BY ?lesionType ?riskFactorType
ORDER BY DESC(?frequency)
```

## CQ13. Which images are associated with patients with a confirmed diagnosis of oral cancer?

```sparql
PREFIX oral: <http://example.org/oral-cancer-ontology#>
PREFIX rdfs: <http://www.w3.org/2000/01/rdf-schema#>

SELECT DISTINCT
       ?patient
       ?image
       ?lesion
       ?label
WHERE {

    ?image oral:belongsToPatient ?patient ;
           oral:hasClinicalDiagnosis ?lesion .

    ?lesion rdfs:label ?label .

    FILTER(
        REGEX(
            LCASE(STR(?label)),
            "oral cancer|cancer|carcinoma"
        )
    )
}
ORDER BY ?patient
```

## CQ14. How do the dataset categories (Normal, Benign, OPMD, and Malignant) relate to clinical lesion diagnoses?

```sparql
PREFIX oral: <http://example.org/oral-cancer-ontology#>
PREFIX rdf: <http://www.w3.org/1999/02/22-rdf-syntax-ns#>

SELECT
       ?category
       ?diagnosisType
       (COUNT(*) AS ?frequency)
WHERE {

    ?image oral:hasCategory ?category ;
           oral:hasClinicalDiagnosis ?diagnosis .

    ?diagnosis rdf:type ?diagnosisType .

    FILTER(?diagnosisType != oral:Lesion)
}
GROUP BY ?category ?diagnosisType
ORDER BY ?category DESC(?frequency)
```

## CQ15. Which diagnoses are most frequently associated with a given risk factor?

```sparql
PREFIX oral: <http://example.org/oral-cancer-ontology#>
PREFIX rdf:  <http://www.w3.org/1999/02/22-rdf-syntax-ns#>

SELECT
       (STRAFTER(STR(?riskFactorType),"#") AS ?RiskFactor)
       (STRAFTER(STR(?diagnosisType),"#") AS ?Diagnosis)
       (COUNT(DISTINCT ?patient) AS ?Patients)
WHERE {

    ?patient oral:hasRiskFactor ?riskFactor ;
             oral:hasMedicalImage ?image .

    ?image oral:hasClinicalDiagnosis ?diagnosis .

    ?riskFactor rdf:type ?riskFactorType .
    ?diagnosis rdf:type ?diagnosisType .

    FILTER(?diagnosisType != oral:Lesion)
}
GROUP BY ?riskFactorType ?diagnosisType
ORDER BY DESC(?Patients)
```
